# Claude Code — Project Rules

## Communication
- Be terse. Point-to-point replies only.
- No trailing summaries, no "what I did" narration.
- No multi-paragraph explanations unless asked.

## Token / Context Management
- Auto-suggest `/compact` when context reaches ~70%.
- Prompt user to open a new chat if the topic drifts from the current task.

## Models (guidance)
- Planning/thinking: use best available model (Opus).
- Code writing/editing: default (Sonnet) is fine.
